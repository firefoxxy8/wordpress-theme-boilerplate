# WordPress Theme Template

## Features
* function
* index
* header
* footer
* page
* singe
* content
* author archives
* date archives
* category and tag archives
* 404
* full width
* contact
* widgets

# Requirements
* [Node.js](https://nodejs.org/)

# Installation
* Run `npm install`

## Usage
* Development: `npm run dev`
* Production: `npm run build`
